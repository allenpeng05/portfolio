Projects

Premier League Match Predictor(Go Brighton)

A data pipeline project that predicts English Premier League match outcomes. I scraped over 10 seasons of data using Selenium and BeautifulSoup and cleaned and processed it with pandas. I engineered features such as rolling_averages and categorical encodings, and trained a Random Forest classifier in scikit-learn with 52% precision.

VR Computer Vision Pipeline

Built in Python with a pretrained Yolov7 model for object detection. I wrote scripts to parse the bounding box outputs and tracked the horizontal movement of a target object across frames to estimate distance, with the assumption that larger shifts in position implied closer proximity.

BruinClassPals

A full-stack React app that helps UCLA students find classmates in their classes. Built with MongoDB, Node.js, and Tailwind. Includes search filters, friends list, and login state management.
